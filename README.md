## ДЗ 1 — 8-входовой мультиплексор (Logisim)

Логическая схема 8-входового мультиплексора, реализованная в Logisim. По трёхразрядному адресному коду A2A1A0 выбирается один из восьми информационных входов M0–M7 и передаётся на выход. Схема построена на элементах NAND, NOR и NOT.

**Файлы:** `multipleksor.circ`, `Edakina_Mariia__var1.pdf`

## ДЗ 2 — Модуль `alu_register` (Verilog), вариант 3

RTL-модуль на Verilog, выполняющий четыре арифметические операции над двумя операндами (сложение, сравнение, сдвиг, НЕ-ИЛИ) и сохраняющий результат в регистре. Тактируется по положительному фронту, сброс синхронный. Покрыт testbench с временными диаграммами в GTKWave.

**Файлы:** `alu_register.v`, `alu_register_tb.v`, `homework2_v3.pdf`

### Запуск

```bash
iverilog alu_register.v alu_register_tb.v
./a.out
gtkwave dump.vcd
```
